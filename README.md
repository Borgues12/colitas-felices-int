# CFELIZ

## Sistema web para la gestión de refugios de animales

CFELIZ es una aplicación web desarrollada con **ASP.NET, C#, SQL Server y JavaScript** orientada a centralizar la gestión administrativa de un refugio de animales sin fines de lucro.

El sistema permite organizar información de mascotas rescatadas, gestionar procesos de adopción y facilitar el manejo interno del refugio.

## Objetivo

El objetivo del proyecto es **donar esta plataforma al refugio "Colitas Felices"**, proporcionando una herramienta digital que apoye su gestión diaria.
El sistema está pensado para ser desplegado en **Azure for Nonprofits**.
